Projects

Chi2

  • I am a lead developer in Noisr, a non profit organization that specializes in generating true randomness from physical phenomena for scientific and for real life applications. More specifically, I am the lead developer of Noisr's Chi2 project, where we aim to create a library in C++ that will become the standard for testing the quality of random number generators by implementing statistical tests that are outlined in specific suites such as the US National Institute of Science and Technology (NIST)'s suite. The project will be published further down the line on my GitHub profile that can be accessed by clicking on the Contact section.
Problem of perfect k-partition

  • In Python, I implemented the problem of determining if a given non empty list of positive values has or does not have a perfect k partition, using the evolutionary programming paradigm. The project is available in my GitHub profile that can be accessed by clicking on the Contact section.
